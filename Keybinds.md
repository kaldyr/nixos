# Applications
Super + q          Terminal
Super + Shift + q  Terminal, but floating
Super + Ctrl  + q  Terminal, but pseudo tiled
Super + r          Run application dialog
Super + u          Color picker (copies hex code to clipboard on mouse click)
Super + c          Calculator

# Screen Capture
Super + p          Take screenshot (drops in ~/Pictures/Screenshots/)
Super + Shift + p  Take screenshot of region (opens with swappy)
Super + Alt   + p  Run OCR on region of screen (copies text into clipboard)
Super + Shift + r  Video record screen to gif (may not work, haven't tried in a while)

# Media
Super + g          Use yt-dlp to open urls in clipboard and play in mpv

# Notifications
Super + n          Close notification
Super + Shift + n  Re-open last notification
Super + Alt   + n  Context of notifcation

# Window management
Super + x          Kill active window
Super + SHIFT + x  Exit Hyprland exit
Super + w          Toggle floating
Super + o          Toggle pseudo tiling
Super + s          Toggle screen split method
Super + f          Toggle window fullscreen

# Focus windows
Mouse click window
Super + h          Focus window left
Super + j          Focus window down
Super + k          Focus window up
Super + l          Focus window right

# Move windows
Super + Left Mouse + Drag
Super + Shift + h  Move window left
Super + Shift + j  Move window down
Super + Shift + k  Move window up
Super + Shift + l  Move window right

# Resize windows
Super + Right Mouse + Drag (closest to corner or edge you want to modify)
Super + Alt   + h  Resize window narrower
Super + Alt   + j  Resize window taller
Super + Alt   + k  Resize window shorter
Super + Alt   + l  Resize window wider
Super + Ctrl  + h  Resize window narrower (big jump)
Super + Ctrl  + j  Resize window taller (big jump)
Super + Ctrl  + k  Resize window shorter (big jump)
Super + Ctrl  + l  Resize window wider (big jump)

# Workspaces
Super + Mouse Wheel Up          Switch to workspace left
Super + Mouse Wheel Down        Switch to workspace right
Super + .          Switch to workspace right
Super + ,          Switch to workspace left
Super + .          Switch to workspace right
Super + # (1-0)    Switch to workspace #
Super + Shift + # (1-0) Move active window to workspace #

# System management
Super + SHIFT + s  Lock the screen
Super + ALT   + s  Sleep system
